### Austin Animal Shelter Exploratory Analysis
**Description:** The goal of this project was to find trends within Intake Vs. Outcome data for animal shelters in Austin, Texas that could better animal welfare and improve shelter conditions by forecasting resource/staffing demands. The dataset contained 173812 rows that needed to be cleaned and joined in SQL, matching outcomes to their corresponding inputs. This data was visualized in Looker Studio and presented in PowerPoint.

**Source Data:** [Intake Data](https://data.austintexas.gov/Health-and-Community-Services/Austin-Animal-Center-Intakes-10-01-2013-to-05-05-2/wter-evkm/explore/query/SELECT%0A%20%20%60animal_id%60%2C%0A%20%20%60name%60%2C%0A%20%20%60datetime%60%2C%0A%20%20%60datetime2%60%2C%0A%20%20%60found_location%60%2C%0A%20%20%60intake_type%60%2C%0A%20%20%60intake_condition%60%2C%0A%20%20%60animal_type%60%2C%0A%20%20%60sex_upon_intake%60%2C%0A%20%20%60age_upon_intake%60%2C%0A%20%20%60breed%60%2C%0A%20%20%60color%60/page/aggregate) / [Outcome Data](https://data.austintexas.gov/Health-and-Community-Services/Austin-Animal-Center-Outcomes-10-01-2013-to-05-05-/9t4d-g238/data_preview)<br>
